Ubuntu XRDP 安装，Windows上连接，以及点击连接后没反应和黑屏解决

<!-- outline-end -->

## 安装

```
sudo apt-get install xrdp
```

开箱即用（并不完全）

## 本地连接

Windows：

在开始菜单搜索 `remote` ，输入ubuntu ip地址，按下回车

看见菜单，Session选择Xorg，输入用户名和登录密码

## 按下回车之后没反应解决办法

见 [Issue](https://github.com/neutrinolabs/xrdp/issues/2321)
